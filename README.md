# Ajax Finder

## Overview

GUI/CLI tool which finds **Ajax** / **XMLHttpRequest** / **Axios** / **fetch** from page URL.


## How to use CLI

- `$ node ajax_finder_cli [URL] [n]`

  - URL: Page URL

  - n : text size(default: 100)


## How to use GUI

- `$ node app`
