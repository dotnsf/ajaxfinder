# Ajax Finder

## Overview

CLI tool which finds **Ajax** / **XMLHttpRequest** / **Axios** / **fetch** from page URL.


## How to use

- `$ node app [URL] [n]`

  - URL: Page URL

  - n : text size(default: 100)
